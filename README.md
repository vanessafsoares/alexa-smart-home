# Controlling Mac or Linux with Alexa Smart Home #

List of shell commands to sync with TRIGGERcmd for Alexa Echo Dot.

### Installation: ###

* MacOS or Linux
* Internet connection

### Usage: ###

``` 
bash open-applications.sh

bash kill-applications.sh

```
